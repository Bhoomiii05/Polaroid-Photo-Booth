📸 AI Polaroid Photo Booth

🚀 Overview

A web-based **Polaroid Photo Booth** that captures photo strips using your camera with filters and AI effects.



✨ Features

* 📷 Capture **3-photo strip with countdown**
* 🎨 Filters: Grayscale, Sepia, Vintage, Cool, Warm, Edge, Blur
* 🤖 Face detection + smart background blur
* 🎨 3 Polaroid border colors (White, Pink, Blue)
* 💾 Save photo strip directly to your device
* 📱 Works on mobile (Chrome) — can capture & download images


🛠 Tech Stack

HTML, CSS, JavaScript, TensorFlow.js


 ▶️ How to Run

Open `index.html` in browser → Allow camera → Capture photos

Built based on the trending **Polaroid photo booth concept**, designed to work even on mobile devices.
